# AI Classification Project

## Project Overview
This project is a basic Artificial Intelligence Classification System developed using Python and Machine Learning.

The system uses the famous Iris Flower Dataset to train and test a machine learning model using the K-Nearest Neighbors (KNN) classification algorithm.

The project demonstrates the fundamentals of supervised learning, including dataset handling, model training, testing, prediction, and accuracy evaluation.

---

## Features
- Load and analyze dataset
- Split data into training and testing sets
- Train AI classification model
- Predict flower categories
- Display model accuracy
- Generate classification report

---

## Technologies Used
- Python
- Pandas
- Scikit-learn

---

## Machine Learning Algorithm
### K-Nearest Neighbors (KNN)
KNN is a supervised learning classification algorithm used to predict the category of data based on the nearest training examples.

---

## Dataset Used
### Iris Flower Dataset
The dataset contains flower measurements such as:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The AI model predicts the flower type:
- Setosa
- Versicolor
- Virginica
